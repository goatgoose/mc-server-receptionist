# mc-server-receptionist

Proxies incomming Minecraft client connections, starting the real server dynamically only when a player attempts to join.

Part of the Goatcraft suite of software. See also:
- [mc-discord-sync](https://github.com/goatgoose/mc-discord-sync)
- [mc-watcher](https://github.com/goatgoose/mc-watcher)
